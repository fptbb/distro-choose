# DistroChoose

A quiz-based Linux distribution recommender. Answer questions about your use case, experience level, philosophy, customization preferences, hardware, and update preferences to find the perfect distro.

This was heavily done with data taken from AI as an experiment to see how well it would work.

It's also a fork from [NeoValues](https://neomediavalist.github.io/neovalues/results.html)

Report issues or contribute on the [DistroChoose GitHub Repository](https://github.com/fptbb/distro-choose).
