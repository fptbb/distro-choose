Updates & Changelog

v1.2 — The App Experience Update

  • Dynamic Map Navigation: The Distro Map is now easily accessible from the main menu! It's been completely rebuilt to smoothly fill modern screens and features powerful new Pan & Zoom graph interaction (mouse wheel/pinch-to-zoom).
  • Massive Icon Overhaul: Swapped out over 50+ generic quiz placeholder shapes with a beautiful, fully customized library of crisp Lucide-powered vector graphics across every single category.
  • Seamless Interlinking: Score Profile traits and "Why this distro?" reasoning bullets are now fully clickable hyperlinks, allowing users to deeply cross-reference their results and definitions across the app.
  • Unified Return-to-Home functionality: You can now intuitively click any main page title to seamlessly return to the homepage without relying on the browser's native back button.
  • Polished Data Visualization: Map axes correctly auto-format to display 0-100% ranges, and backend HTML paths and title SEO tags have been rigorously unified.

v1.1 — The Interactive Update

  • Interactive Distro Map: Explore distributions plotted on a 2D map with 12 custom dimensions like "Beginner Friendliness" vs "Customization Depth", featuring distro logos with hover effects.
  • Deep Dive Descriptions: Every distro and quiz value now has a dedicated page with detailed information, score profiles, and recommendations.
  • Animated Score Profiles: Visual bar charts showing exactly how strongly a distro aligns with specific traits.
  • "Why this distro?" Breakdown: The results page now highlights exactly which of your quiz answers led to the top distro recommendation.
  • Progression & Sharing: Added a sleek progress bar to the quiz and a "Share Results" button to easily copy your matches to the clipboard.
  • Updated Visuals: Polished icons with a unified aesthetic featuring distro logos.
