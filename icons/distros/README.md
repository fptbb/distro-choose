# Distro Icons

Place distro logo images here using the **slug name** from the data files.

## Naming Convention

The slug is generated from the distro name:
- Lowercase
- Replace spaces and special characters with `_`
- Remove consecutive underscores

## Examples

| Distro Name | File |
|---|---|
| Ubuntu | `ubuntu.png` |
| Linux Mint | `linux_mint.png` |
| Pop!_OS | `pop_os.png` |
| Arch Linux | `arch_linux.png` |
| openSUSE Tumbleweed | `opensuse_tumbleweed.png` |
| elementary OS | `elementary_os.png` |
| Fedora Workstation | `fedora_workstation.png` |
| NixOS | `nixos.png` |
| Kali Linux | `kali_linux.png` |
| TrueNAS SCALE | `truenas_scale.png` |

## Source

DistroWatch icon URLs follow this pattern:
```
https://distrowatch.com/images/yvzhuwbpy/<distrowatch_id>.png
```

Check each distro's page on distrowatch.com to find the correct ID.

## Size

Recommended: 100x100px PNG with transparent background.
